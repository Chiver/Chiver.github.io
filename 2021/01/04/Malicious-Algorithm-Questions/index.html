[4-Hard] Median of Tow Sorted Arrays

Given two sorted arrays nums1 and nums2 of size m and n respectively, return the median of the two sorted arrays.

Require that the time complexity as O(log(m+n)).

Example 1:

Solution:

Make the shorter array as the one to perform binary search. The central idea of this algorithm is that we can pre-calculate the total length of the sum of the upper left half and the lower left half of the arrays, which is (m+n+1)//2. And then, performing binary search on the upper (shorter) array to calculate the corresponding length of the lower index by (m+n+1)//2 - upper_index, where the upper_index is the index of the binary search. [76-Hard] Minimum Window Substring

Given two strings s and t, return the minimum window in s which will contain all the characters in t. If there is no such window in s that covers all characters in t, return the empty string "".

Note that If there is such a window, it is guaranteed that there will always be only one unique minimum window in s.

Example 1:

Solution:
This is yet another sliding window question. Using the framework, we need to keep track of the following elements:

  • left, right pointers
  • "need" mapping, "window" mapping (where need mapping keeps track of the remaining element counts for the window to fulfill the requirement of t, and the window keeps track of all the counts in the window)
  • valid variable which keeps track of the number of the current fulfilled characters (this could also be derived from need but just a syntatic sugar that omits extra computation)

[567-Medium] Permutation in String

Given two strings s1 and s2, write a function to return true if s2 contains the permutation of s1. In other words, one of the first string’s permutations is the substring of the second string.

Example 1:

Solution:

This is yet another slide window question. We have the following steps:

  1. Move right pointer if : right < Len(s2)
  2. Mode left pointer if : the current substring [left, right) contains the permutation
  3. Exit / Update condition: if right-left == len(s1) and [left, right) satisfies the permutation condition
